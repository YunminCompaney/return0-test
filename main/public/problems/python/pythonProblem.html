Python(파이썬) 영역

단답형

  1. 다음 코드의 오류를 찾아보세요. (배점: 1점)

  2. 외부 모듈이나 라이브러리를 현재 스크립트로 불러와서 해당 모듈이나 라이브러리의 기능을 사용할 수 있게 해주는 키워드는 무엇인가요? (배점: 1점)

  3. 소스 2는 Python로 작성된 프로그램의 소스코드이다. 다음 프로그램을 실행시켰을때, 출력되는 값은? (배점: 2점)

  4. 소스 3은 Python로 작성된 프로그램의 소스코드이다. 이 소스코드에는 오류가 없는가? (예/아니요)로 답하세요. ('아니요'로 답한 경우에는, 오류가 있는 부분을 수정해보세요.) (배점: 2점)

  5. 다음 질문에 답하세요.


    5-1.프로그램을 실행했을 따, n = 20 이라면 출력값은 무엇인가요? (배점:2점)

    5-2.이 프로그램에서 n = x라고 입력했을 때 출력값이 50이상이 되도록 하는 x의 최소값은 무엇인가요? (배점:3점)

  6. 소스 5를 실행했을때, x = 13, y = 7 인 경우 출력되는 값은 무엇인가요? (배점: 3점)

서답형

  1. 1부터 N까지의 합을 계산하는 프로그램을 작성하세요. (N은 사용자 입력) (배점: 3점)

  2. 사용자로부터 𝑛개의 정수를 입력받아, 그 중 소수(Prime Number)만 출력하는 프로그램을 작성하십시오. 소수를 출력한 후, 총 몇 개의 소수가 있는지 출력하세요. (배점: 6점)

  3. 다음 조건을 충족하는 함수를 작성하세요. (배점: 8점)

    1. 두 리스트의 공통 요소를 찾아 반환해야 합니다.
    2. 중복 없이 반환해야 합니다.
    -----------------------------------------------------------------
    # 예: list1 = [1, 2, 2, 3], list2 = [2, 2, 3, 4]
    # 반환값: [2, 3]

서답형 2번 채점 기준

  1. (2점) 소수를 정확히 판별하는 코드를 작성했는가?
  2. (2점) 입력 데이터에서 소수만 출력하는 기능이 구현되었는가?
  3. (1점) 소수의 개수를 올바르게 출력했는가?
  4. (1점) 소수가 없는 경우에 대한 처리와 출력이 올바른가?

서답형 3번 체점 기준

  1. 함수 정의 및 입력 처리 (2점)
      - 함수가 올바르게 정의되고 두 개의 리스트를 입력으로 받을 수 있어야 합니다. (1점)
      - 입력 리스트가 함수 내부에서 올바르게 처리되어야 합니다. (1점)
  2. 공통 요소 찾기 (4점)
      - 두 리스트의 공통 요소를 올바르게 찾을 수 있어야 합니다. (2점)
      - 공통 요소가 중복 없이 처리되어야 합니다. (2점)
  3. 결과 반환 (2점)
      - 결과가 리스트 형태로 반환되어야 합니다. (1점)
      - 반환된 리스트의 요소들이 올바르게 공통 요소를 포함해야 합니다. (1점)
