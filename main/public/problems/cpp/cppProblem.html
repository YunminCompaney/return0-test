2025년 구월중학교 문제지 {프로그래밍 언어(C++) 영역}

아래의 문제들을 해결하세요. 만점: 30점

단답형

  1. 다음 코드의 오류를 찾아보세요. (배점: 1점)

  2. C++ 표준 입출력을 위한 헤더 파일의 이름은 무엇인가요? (입력 형식: <헤더 파일의 이름>) (배점: 1점)

  3. 소스 2는 C++로 작성된 프로그램의 소스코드이다. 다음 프로그램을 실행시켰을때, 출력되는 값은? (배점: 2점)

  4. 소스 3은 C++로 작성된 프로그램의 소스코드이다. 이 소스코드에는 오류가 없는가? (예/아니요)로 답하세요. ('아니요'로 답한 경우에는, 오류가 있는 부분을 수정해보세요.) (배점: 2점)

  5. 다음 질문에 답하세요.


    5-1.위 프로그램을 실행했을 때, n = 10 이라면,출력값은 무엇인가요?(배점:2점)

    5-2.프로그램의 규칙에 따라 n = 20일 때 출력값은 무엇인가요?(배점:2점)

    5-3.이 프로그램에서 n = x라고 입력했을 때 출력값이 50이상이 되도록 하는 x의 최소값은 무엇인가요?(배점:3점)

  6. 소스 5를 실행했을때, 출력되는 값은?(배점: 5점)

서답형

  1. 1부터 N까지의 합을 계산하는 프로그램을 작성하세요. (N은 사용자 입력) (배점: 4점)

  2. 사용자로부터 𝑛개의 정수를 입력받아, 그 중 소수(Prime Number)만 출력하는 프로그램을 작성하십시오. 소수를 출력한 후, 총 몇 개의 소수가 있는지 출력하세요(배점: 8점)

서답형 2번 채점 기준

  1. (2점) 소수를 정확히 판별하는 코드를 작성했는가?
  2. (2점) 입력 데이터에서 소수만 출력하는 기능이 구현되었는가?
  3. (2점) 소수의 개수를 올바르게 출력했는가?
  4. (2점) 소수가 없는 경우에 대한 처리와 출력이 올바른가?
